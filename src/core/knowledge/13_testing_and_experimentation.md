## A/B Testing Fundamentals

An A/B test compares a control against a treatment with a single variable changed. Valid tests require: sufficient sample size (minimum 1,000+ per variation), random assignment, a single variable, and a pre-defined primary metric and success threshold.

Common mistakes: Stopping tests early (peeking problem), testing too many variables at once, not accounting for novelty effect, using the wrong metric, running tests during atypical periods.

## Creative Testing Matrices

For paid media:
- Test one variable at a time: hook vs. hook, angle vs. angle, format vs. format
- Test at least 3-5 variations per variable
- On Meta, Advantage+ distributes traffic automatically
- On TikTok, test 20-30 creative variations per month
- Document winners in a "creative bank" with rationale for why they won

## Lift Tests and Incrementality

Platform-reported ROAS is not incrementality. Platforms over-attribute because they claim credit for conversions that would have happened anyway.

Types of incrementality measurement:
- Holdout tests: Withhold ads from a random subset, measure conversion rate difference
- Geo tests: Run ads in some markets, withhold in others, compare results
- Conversion lift studies: Platform-native (Meta, Google) tests measuring incremental ad impact
- Brand lift studies: Measure changes in awareness, consideration, preference

When to use: When spend exceeds $50K/month, when platform ROAS seems too good, when making major budget reallocations, when evaluating channel expansion.

## Landing Page Testing

Priority order for testing:
1. Headline (biggest impact on first-impression relevance)
2. Hero section/above-the-fold content
3. CTA copy and placement
4. Social proof type and placement
5. Page length (short vs. long)
6. Form fields (fewer = higher completion, lower lead quality)

---

## Budget Scaling Logic

- Increase by no more than 10-20% every 48-72 hours on Meta
- Monitor CAC and ROAS as you scale; efficiency typically decreases at higher spend
- Use "marginal ROAS" concept: the return on the next dollar spent, not the average
- At diminishing returns on one channel, diversify to new channels
