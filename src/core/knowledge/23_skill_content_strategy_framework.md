# Skill Framework: Content Strategy Engine
> Type: Thinking Methodology (Step-by-step)
> When to use: Content calendar generation, pillar planning, content concept ideation
> Confidence: High

---

## Mandatory Thinking Process for Content Strategy

### Step 1: Audience State Mapping
Before generating ANY content plan, categorize the audience into awareness states:
- UNAWARE: Don't know they have a problem → Content goal: pattern interrupt, provoke
- PROBLEM-AWARE: Know the pain, not the solution → Content goal: agitate, validate
- SOLUTION-AWARE: Know solutions exist, evaluating → Content goal: differentiate, prove
- PRODUCT-AWARE: Know this brand, haven't bought → Content goal: overcome objections, urgency
- MOST-AWARE: Customers/fans → Content goal: reinforce, community, advocacy

Every content calendar MUST have posts targeting each state. Ratio depends on brand maturity:
- New brand: 40% Unaware, 30% Problem, 20% Solution, 10% Product
- Established brand: 15% Unaware, 20% Problem, 25% Solution, 25% Product, 15% Most-Aware

### Step 2: Content Pillar Stress Test
For each proposed pillar, it must pass ALL checks:
- [ ] Ties to a JTBD the audience actually has
- [ ] Has a clear awareness state it serves
- [ ] Can generate 10+ unique content pieces (not exhausted in 3 posts)
- [ ] Differentiates from what competitors post (check competitor pillars)
- [ ] Maps to at least one business KPI (awareness, traffic, leads, sales, retention)

Kill any pillar that fails 2+ checks.

### Step 3: Hook Architecture
Every piece of content follows this hierarchy:
1. HOOK (0-3 seconds): Pattern interrupt. Must pass the "thumb-stop test"
2. HOLD (3-15 seconds): Reward the attention. Deliver on the hook's promise
3. PAYOFF (15-60 seconds): Core value. The reason they'll share/save/comment
4. CTA (last 3 seconds): One clear next action. Not "like comment subscribe follow"

Hook types ranked by effectiveness:
- Contrarian statement: "Stop doing X" / "X is dead"
- Specific number: "I spent $47,000 testing this"
- Identity trigger: "If you're a [role] who [behavior]..."
- Open loop: "The thing nobody tells you about X"
- Social proof: "How [known entity] does X"

### Step 4: Format-Platform Matching
Never assign formats randomly. Use this decision tree:
- Complex idea + needs sequence → Carousel
- Emotional story + human face → Reel/Video
- Hot take + conversation starter → Text post
- Data/insight + save-worthy → Infographic/Static
- Behind-the-scenes + authenticity → Story
- Long-form thought leadership → Article/Thread

### Step 5: Engagement Architecture
Design each post for a specific engagement type:
- SAVES: Educational, reference-worthy, "I'll need this later"
- SHARES: Identity-affirming, "this is SO me", tribal
- COMMENTS: Controversial, incomplete, asks for opinion
- CLICKS: Curiosity gap, teaser, "link in bio" worthy
- FOLLOWS: Series content, "follow for part 2", consistent value

---

## Calendar Distribution Rules
- Never post the same pillar twice in a row
- Never post the same format three times in a row
- Alternate between awareness states across the week
- Front-load the week with highest-reach formats (Reels Mon-Wed, Carousels Thu-Fri)
- Weekend = lighter, more personal/BTS content
